# Auction

- 拍卖网站
- 前端 URL: [Auction-vite](https://github.com/Valinaa/Auction-vite)

## Features

1. 基于Spring Boot构建，在master分支采用Apache Shiro作为安全框架
2. 使用Mybatis-Plus提高开发效率
3. 基于Redis的ZSet数据结构构建排行榜
4. 能自行添加商品设置竞拍信息，也能参加竞拍，竞拍时间截至后自动生成排行榜
